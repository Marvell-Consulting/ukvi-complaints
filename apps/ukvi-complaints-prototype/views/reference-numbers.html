Help with reference numbers

Global Web Form (GWF) number

You will have been given a GWF number if you applied for a visa outside of the UK.

Home Office reference number

You will have been given a Home Office reference number if you applied for a visa within the UK. This could be an application or appeal number, an Application Registration Card (ARC) number or a case ID number.

International health surcharge (IHS) number

You will have been given an IHS number if you paid for the Immigration Health Surcharge.
