Help with reference numbers

GWF number

You should have been given a GWF number if you applied for a visa outside of the UK.

Home Office reference number

You should have been given a Home Office reference number if you applied for a visa within the UK. This could be an application or appeal number, an Application Registration Card (ARC) number or a case ID number.

IHS number

You should have been given an IHS number if you paid for the Immigration Health Surcharge.
